隐私政策

1. 信息收集

我们仅收集您在使用本平台广告管理服务时主动提供的信息，包括但不限于：账户注册信息、广告投放配置数据、以及通过 Meta Marketing API 获取的广告成效数据。

2. 信息使用

我们收集的信息仅用于：提供和维护广告管理服务、优化广告投放策略、以及生成广告成效报告。我们不会将您的信息用于任何其他目的。

3. 信息共享

我们不会向第三方出售或分享您的个人信息。我们仅通过与 Meta 平台的 API 交互来执行您授权的广告管理操作。

4. 数据安全

我们采取合理的技术措施保护您的数据安全，包括数据加密传输、访问权限控制等。

5. 数据保留

我们仅在提供服务所必需的期限内保留您的数据。您可以随时要求删除您的账户及相关数据。
